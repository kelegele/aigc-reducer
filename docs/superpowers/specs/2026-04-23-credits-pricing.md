# 积分定价分析

## 成本基础

- **LLM 模型**: Kimi 2.6
- **输入价格**: ¥6.5 / 百万 token
- **输出价格**: ¥27 / 百万 token

### Token 消耗模型

一篇论文走完检测 + 改写流程，原始 token 约被处理 3-4 次：
1. 检测：输入原文 + prompt → 输出评分（输入多、输出少）
2. 改写：逐段输入原文 + prompt → 输出改写文（输入输出约 1:1）

混合比例约 2:1（输入多输出少），**混合均价 ≈ ¥13.3 / 百万 token**

### 中文论文 Token 估算

| 论文类型 | 中文字数 | 约 Token 数 |
|---------|---------|------------|
| 本科短文 | 5,000 字 | ~8,000 |
| 本科论文 | 10,000-15,000 字 | ~16,000-24,000 |
| 硕士论文 | 30,000-40,000 字 | ~48,000-64,000 |

> 1 个中文字 ≈ 1.5-2 token
